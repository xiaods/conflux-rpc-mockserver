# conflux-rpc-mockserver

背景:
-----
OpenRPC是更佳友好的RPC API界面，提供测试Mock，方便开发了解Conflux的API变化

目标：
-----
引更多区块链爱好者参与Conflux社区建设

任务：
----
对应Conflux编写OpenRPC的描述文件，方便开发做conflux rpc api testing


测试用playground: https://playground.open-rpc.org/
